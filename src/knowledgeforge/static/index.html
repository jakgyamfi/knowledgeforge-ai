About KnowledgeForge

KnowledgeForge began as a secure way to capture lessons from cloud security and AI infrastructure work before they disappeared. It preserves private source material, transcribes locally, and uses the AI provider you choose to turn fragments into evolving books, businesses, and projects.

Built by Agyapong Gyamfi, a Cloud Security Engineer focused on securing AI agents, MCP servers, and the cloud infrastructure they depend on.
